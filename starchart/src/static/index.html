Starchart Demo

This is a demonstration, which allows the creation of subdomains and wildcard certificates for the starchart.invalid domain. It will set the appropriate DNS records in PowerDNS via its HTTP API. Everything you see here is being run in docker containers locally in order to demonstrate the setup.

Subdomain

Choose a hostname to create at hostname.username.starchart.invalid. You can either use an A Record to an IP Address or a CNAME Record to a Domain.

Certificates

NOTE: this certificate is generated using Pebble, a mock implementation of Let's Encrypt, and should not be used in production.
